# 이우 전화 감지기 (Android)

병원 전용 핸드폰에 설치하는 보조 앱.

## 하는 일
1. **전화 수신 감지** — 병원 폰으로 전화 오면 번호를 백엔드에 POST → 데스크탑 환자목록에 토스트 표시
2. **환자 주소록 동기화** — Firebase 환자 데이터를 폰 주소록 ("이우병원" 그룹) 에 15분마다 동기. 전화 올 때 표준 통화 화면에 환자 이름·진단 자동 표시

## APK 받기 (간편)
1. GitHub 의 **Actions** 탭 클릭 → "Android APK Build" 워크플로우 → 최신 성공한 run 클릭
2. 페이지 하단 **Artifacts** 의 `ewoo-call-detector-debug` 다운로드 → ZIP 안에 `app-debug.apk`
3. 폰으로 APK 전송 (USB/카톡/메일 무관) → 설치 (출처를 알 수 없는 앱 허용 필요)

## 첫 실행 설정
1. 앱 실행
2. **권한 요청** 버튼 → 4가지 권한 (전화상태/통화기록/주소록 읽기·쓰기) 모두 허용
3. **공유 시크릿** 입력란에 관리자에게 받은 시크릿 키 붙여넣기
4. **설정 저장** 버튼
5. **지금 환자 동기화** 버튼 → 잠시 후 폰 주소록 → 이우병원 그룹에 환자 표시 확인

## 배포·업데이트
- 코드 수정 후 `git push` → GitHub Actions 자동 빌드 (~5분) → Artifacts 에 새 APK
- 사용자는 새 APK 다운로드 후 폰에 재설치 (기존 데이터 유지됨)

## 주요 동작
- **15분 주기 동기화** — WorkManager 가 자동 실행. 새 환자·전화번호·진단명 변경 자동 반영
- **부팅 후 자동 시작** — 폰 재부팅해도 동기 작업 자동 재개
- **폰 분실 시** — 주소록의 "이우병원" 그룹 한 번에 삭제 (다른 개인 연락처는 무영향)
