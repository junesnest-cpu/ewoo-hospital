### P1 — 환자 마스터 로컬 캐시 (경량 sync 에서도 전체 환자 매칭)

- **부하**: RPi CPU +50~100ms, 메모리 +30MB, 사용자 UI 영향 0

### P2 — 매칭 우선순위 재설계
4단계 강→약:
1. `chartNo` 직접

4. `birthYear` + baseName 유일

`baseName`: `신)` 접두어 + 끝자리 숫자 + whitespace 제거 (동명이인·표기변형 내성).

### P3 — 오연결 의심 탐지

- `severity`: 2개 이상 필드 불일치 or baseName 불일치 = **high**, 1필드만 = **medium**
